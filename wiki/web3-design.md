# Web3 交易所设计规律

> 从 OKX / Binance / Bybit / Coinbase / Kraken / Gate.io / HTX 提取

## 五条铁律

### 1. 涨跌色是第一优先级
整个界面围绕涨（绿 #0ECB81）跌（红 #F6465D）构建。品牌色必须跟涨跌色拉开距离。

### 2. 等宽数字排版
价格必须用等宽字体，小数点对齐，变化时闪烁高亮。

### 3. 深色模式为默认
交易者长时间盯盘，深色减少视觉疲劳。

### 4. 极致信息密度
交易页面一屏 50+ 数据点。不追求留白，追求信息效率。

### 5. 实时数据
WebSocket 推送，价格变化不能导致布局抖动。
